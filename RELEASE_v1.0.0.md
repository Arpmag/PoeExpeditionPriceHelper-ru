## Что нового в v1.0.0

Первая публичная сборка **русской** версии оверлея цен.

- **OCR на русском** (`rus` Tesseract) для клиента Path of Exile 2 на русском языке
- **Словари RU→EN** для предметов Экспедиции, рун, валюты и сплавов (данные [poe2db.tw/ru](https://poe2db.tw/ru))
- **Рунотворческие комбинации** — цены на `Сфера астромантии`, `Руна мощи`, `Большая руна искусности`, сплавы и др.
- **Интерфейс на русском** — калибровка, статус, горячие клавиши
- Все функции оригинала: live poe.ninja, стеки, неогранённые камни по уровню, трей, multi-monitor, настраиваемые hotkeys

## Установка

1. Скачайте `PoeExpeditionPriceHelper-v1.0.0-win-x64.zip` ниже.
2. Распакуйте куда угодно.
3. Запустите **Start.cmd**. .NET устанавливать не нужно.

Подробная инструкция — в **README.md** и **README.html** внутри архива.

> **Английский клиент?** Используйте оригинал:  
> https://github.com/pedro-quiterio/PoeAncientsPriceHelper/releases

> Windows SmartScreen может предупредить (unsigned) — **Подробнее → Выполнить в любом случае**.
