# Встановлення «Тверезі дні» на iPhone

## Швидкий спосіб (без Mac)

1. Відкрий у **Safari** на iPhone:
   **https://arpmag.github.io/PoeExpeditionPriceHelper-ru/**
2. **Поділитися** → **На Початковий екран** → **Додати**

Додаток з’явиться на головному екрані як звичайний.

## Якщо посилання не працює

У репозиторії GitHub: **Settings → Pages → Build and deployment → Source: GitHub Actions**.

Після злиття змін у `main` деплой запуститься автоматично (1–2 хв).
