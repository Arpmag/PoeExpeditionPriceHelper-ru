# Тверезі дні — iOS / iPhone

Додаток для відстеження тверезих днів. Інтерфейс українською.

## Найпростіший спосіб — встановити на iPhone за 1 хвилину

**Не потрібен Mac і Xcode.** Додаток працює як веб-додаток (PWA) і встановлюється прямо на головний екран iPhone.

### Крок 1. Відкрий посилання на iPhone

Після злиття PR у `main`, додаток буде доступний за адресою:

**https://arpmag.github.io/PoeExpeditionPriceHelper-ru/**

(Якщо сторінка ще не відкривається — увімкни GitHub Pages: репозиторій → Settings → Pages → Source: **GitHub Actions**)

### Крок 2. Додай на головний екран

1. Відкрий посилання в **Safari** на iPhone
2. Натисни кнопку **Поділитися** (квадрат зі стрілкою вгору)
3. Прокрути вниз і обери **«На Початковий екран»**
4. Натисни **«Додати»**

Готово! Іконка «Тверезі дні» з’явиться на головному екрані. Додаток працює офлайн, дані зберігаються на телефоні.

## Нативний iOS-додаток (через Xcode на Mac)

Якщо є Mac з Xcode — можна встановити нативну версію на iPhone.

### Вимоги

- Mac з **Xcode 15+**
- iPhone з **iOS 17+**
- Безкоштовний **Apple ID** (або платний Apple Developer за $99/рік)

### Кроки

1. Клонуй репозиторій на Mac
2. Відкрий `ios/SobrietyTracker/SobrietyTracker.xcodeproj` у Xcode
3. У Xcode: **Signing & Capabilities** → увімкни **Automatically manage signing**
4. Обери свій **Team** (увійди Apple ID: Xcode → Settings → Accounts)
5. Зміни **Bundle Identifier** на унікальний, наприклад `com.твоєімʼя.sobrietytracker`
6. Підключи iPhone кабелем, обери його як цільовий пристрій
7. На iPhone: **Налаштування → Основні → VPN і керування пристроєм** → довіряй розробнику
8. Натисни **Run** (⌘R) у Xcode

> **Примітка:** з безкоштовним Apple ID додаток працює ~7 днів, потім потрібно знову запустити з Xcode. З платним акаунтом — до року.

## Структура

```
web/                    ← PWA для iPhone (GitHub Pages)
ios/SobrietyTracker/    ← нативний SwiftUI-додаток
```

## Локальний запуск PWA

```bash
cd web
python3 -m http.server 8080
```

Відкрий `http://localhost:8080` у браузері.
