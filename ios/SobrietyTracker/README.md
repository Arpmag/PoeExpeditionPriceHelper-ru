# Тверезі дні — iOS додаток

Додаток для iPhone, який рахує кількість тверезих днів. Інтерфейс українською мовою.

## Можливості

- **Лічильник днів** — великий головний лічильник з датою початку
- **Детальна статистика** — тижні, години, хвилини
- **Досягнення** — міленстоуни: 1 день, тиждень, місяць, 90 днів, рік і більше
- **Прогрес до наступної цілі** — скільки днів залишилось
- **Налаштування** — зміна дати початку або скидання лічильника
- **Збереження даних** — дата зберігається локально на пристрої

## Вимоги

- macOS з Xcode 15+
- iPhone з iOS 17+ (або симулятор)

## Як запустити

1. Клонуй репозиторій
2. Відкрий `ios/SobrietyTracker/SobrietyTracker.xcodeproj` у Xcode
3. Обери свій iPhone або симулятор
4. Натисни **Run** (⌘R)

## Структура проєкту

```
SobrietyTracker/
├── SobrietyTrackerApp.swift    # Точка входу
├── Models/
│   └── SobrietyStore.swift     # Логіка та збереження даних
├── Views/
│   ├── ContentView.swift       # Маршрутизація екранів
│   ├── OnboardingView.swift    # Перший запуск
│   ├── HomeView.swift          # Головний екран
│   ├── MilestonesView.swift    # Досягнення
│   └── SettingsView.swift      # Налаштування
├── Components/
│   └── StatCard.swift          # Картка статистики
└── Helpers/
    └── UkrainianPlural.swift   # Правильні форми слів (день/дні/днів)
```

## Перший запуск

При першому відкритті додаток запитає дату початку тверезості. Можна обрати «Почати з сьогодні» або вказати минулу дату.

## Ліцензія

Особистий проєкт. Використовуй вільно.
